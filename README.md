# 카톡 답장 도우미 💬

상사와의 카톡 대화를 분석하여 상황에 맞는 답장 초안을 생성해주는 AI 도우미입니다.

## 기능

- 📸 카톡 캡쳐 이미지 OCR 자동 추출
- 🎯 톤 선택 (매우 공손 / 중립적 / 적극적)
- 📋 복사 가능한 답장 초안 생성

## 실행 방법

```bash
UPSTAGE_API_KEY=your_api_key node index.js
```

브라우저에서 http://localhost:3000 접속

## 사용 API

- [Upstage Solar](https://www.upstage.ai/) - Chat & OCR
